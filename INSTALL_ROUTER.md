# Instalação do React Router

Para que o blog funcione corretamente, você precisa instalar o `react-router-dom`.

## Instruções:

Execute o seguinte comando no terminal:

```bash
npm install react-router-dom
```

Ou, se tiver problemas de permissão:

```bash
sudo npm install react-router-dom
```

Após a instalação, o blog estará totalmente funcional com:
- Página de listagem de artigos (`/blog`)
- Páginas individuais de artigos (`/blog/:slug`)
- Navegação integrada no Navbar
